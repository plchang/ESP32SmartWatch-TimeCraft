# ESP32SmartWatch-TimeCraft

作業系統: FreeRTOS

硬體: ESP32 Start Kit、 ESP32 S、1.28寸 IPS圓形液晶顯示幕 LCD模組 GC9A01驅動 SPI串口 240*240 有資料、MAX30102血氧濃度手腕心率脈搏檢測心跳感測器模組 STM32感測器、電測量 AD8232 脈搏 心臟 心電圖 套件 相容 Duinofun

軟體: microPy
